# Stic'K

## Project Description

Stic'K is a simple and colorful note-taking app designed to streamline task management for a K-beauty supply company. With a virtual sticky wall, it enables users to create personalized to-do lists for each buyer, add, edit, and delete tasks, and efficiently manage a customized pipeline in one display.

The app also serves as a general, user-friendly, and colorful note-taking tool that can be used by anyone, just like a virtual sticky note wall.

## Installation

Stic'K is accessible online via the following link: [Stic'K App](https://stickapp-jhs7a28zk-imaligrace.vercel.app/)

## Usage

### Internal Company Usage:

- Create a new buyer entry with a to-do note for each buyer.
- Edit and delete tasks for each buyer.
- View all buyers' sticky notes and their tasks for quick overviews.

### General Note-Taking:

Stic'K is also designed to be a versatile, colorful note-taking app suitable for a wide range of users. It can be used by anyone with no required log-ins or additional sign-up.
